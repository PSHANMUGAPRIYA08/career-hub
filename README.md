# 🚀 Career Hub

Career Hub is a web-based career management application developed using HTML, CSS, and JavaScript.

It helps users manage tasks, track job applications, monitor progress, and stay organized during their job search journey.

## ✨ Features

### 📋 Task Manager

* Add tasks
* Edit tasks
* Delete tasks
* Set priorities
* Set due dates
* Track completion progress

### 💼 Job Tracker

* Add job applications
* Edit applications
* Delete applications
* Track application status

### 📊 Analytics Dashboard

* Total tasks
* Completed tasks
* Pending tasks
* Total job applications

### 👤 Profile Management

* Save profile information
* Edit profile details
* Persistent data using Local Storage

### 🏆 Achievement System

* First Task Added
* First Job Application
* Task Completion Milestones
* Interview Achievements

### 🎨 Additional Features

* Dark/Light Theme Toggle
* Daily Motivation Quotes
* Live Clock
* PDF Report Export
* Responsive Design
* Login System

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage
* jsPDF
